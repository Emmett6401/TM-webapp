### 4. 해야할 숙제는 
1. https://teachablemachine.withgoogle.com/ 에 접속하세요
2. 로그인을 하면 자신의 프로젝트가 남아 있습니다.
3. 이미지 프로젝트를 시작해 보세요 
4. 참고하기 위해서 project.tm을 다운로드 해서 활용 할 수 있습니다.
5. index.html을 수저애서 좀더 예쁜 UI를 직접 구성해 보아요
6. 결과물을 자신의 프로필 페이지에 연결해 보세요
